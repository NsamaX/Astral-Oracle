<h1 align="center">✨ Astral Oracle ✨</h1>

<p align="center">Welcome to <strong>Astral Oracle</strong>, a mystical web application that provides tarot card readings using advanced AI technology. Dive into the world of tarot and explore the insights it offers for your life's journey. 🌌</p>

<h2>🌟 Live Demo</h2>
<p>You can experience the application live at: <a href="https://astral-oracle.vercel.app/" target="_blank" rel="noopener noreferrer">Astral Oracle</a></p>

<h2>🛠️ Features</h2>
<ul>
    <li>🔮 <strong>Tarot Card Reading:</strong> Draw single or multiple cards to gain insights about your queries.</li>
    <li>🖱️ <strong>Interactive User Interface:</strong> Click on the cards to shuffle and reveal their meanings.</li>
    <li>🤖 <strong>AI Predictions:</strong> Get personalized responses from an AI Tarot reader based on the cards drawn.</li>
    <li>📜 <strong>History Tracking:</strong> Save your past readings for future reference.</li>
    <li>🍪 <strong>Cookie Consent Management:</strong> Manage user cookie preferences for enhanced privacy.</li>
</ul>

<h2>💻 Technologies Used</h2>
<ul>
    <li>⚛️ <strong>React:</strong> Frontend framework for building the user interface.</li>
    <li>📚 <strong>Groq SDK:</strong> For AI-powered tarot reading predictions.</li>
    <li>🃏 <strong>Tarot API:</strong> To fetch tarot card data and meanings.</li>
    <li>🌐 <strong>JavaScript:</strong> Primary programming language used for development.</li>
    <li>🎨 <strong>CSS:</strong> For styling the application.</li>
</ul>

<h2>🔧 Installation</h2>
<p>To run this project locally, follow these steps:</p>

<pre><code>1. Clone the repository:</code></pre>
<pre><code>   git clone https://github.com/NsamaX/Astral-Oracle.git</code></pre>

<pre><code>2. Navigate to the project directory:</code></pre>
<pre><code>   cd Astral-Oracle</code></pre>

<pre><code>3. Install the dependencies:</code></pre>
<pre><code>   npm install</code></pre>

<pre><code>4. Create a .env file in the root directory and add your Groq API key:</code></pre>
<pre><code>   VITE_GROQ_API_KEY=your_api_key_here</code></pre>

<pre><code>5. Start the application:</code></pre>
<pre><code>   npm run dev</code></pre>

<p>Open your browser and visit <a href="http://localhost:5173" target="_blank" rel="noopener noreferrer">http://localhost:5173</a>.</p>

<h2>🧙 Usage</h2>
<ol>
    <li>Select the number of cards you wish to draw (1, 3, or 5).</li>
    <li>Enter your question in the provided input box.</li>
    <li>Click on the cards to reveal their meanings and receive insights based on your question.</li>
</ol>
